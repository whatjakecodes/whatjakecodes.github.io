# whatjakecodes.github.io

## Run site locally

Install node.js via chocolatey
```zsh
# in windows git bash:
choco install nodejs.install --force -y

## ensure folder created
mkdir $USERPROFILE\AppData\Roaming\npm
```

Open command line and run:

```zsh
cd whatjakecodes.github.io
npx http-server .
```
